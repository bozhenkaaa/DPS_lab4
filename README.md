# Лабораторна робота: Фазові портрети для двовимірних систем

## Опис

Ця лабораторна робота дозволяє моделювати та візуалізувати фазові портрети для двовимірних динамічних систем:
- Лінійних систем (наприклад, центр)
- Однорідних нелінійних систем (наприклад, промені)

---

## Вимоги

- Python 3.8+
- numpy
- matplotlib

Встановити залежності:
```bash
pip install numpy matplotlib
```

---

## Запуск статичного графіка (Matplotlib)

1. Скопіюйте код для лінійної або однорідної системи у файл, наприклад, `phase_portrait.py`.
2. Запустіть:
    ```bash
    python main1.py
    ```
3. Відкриється вікно з фазовим портретом.

## Приклади систем

### Лінійна система (центр)

$$
\begin{cases}
x' = -y \\
y' = x
\end{cases}
$$

### Однорідна система (промені)
$$
\begin{cases}
x' = x^2 - y^2 \\
y' = 2xy
\end{cases}
$$

---

## Функціонал

- Побудова фазового портрету для вибраної системи
- Візуалізація напрямків руху (векторне поле)
- Відображення особливих точок (рівноваг)
